# Google Maps Business Scraper

Web-based tool to scrape business data from Google Maps and export to Excel.

## Setup

```bash
npm install
pip install playwright && playwright install chromium
```

## Run

```bash
node server.js
```

Open http://localhost:3000

## Features

- Select US state, pick cities
- Scrapes company name, phone numbers, emails, website
- Real-time progress tracking
- Cancel anytime, download partial results
- Exports to Excel
